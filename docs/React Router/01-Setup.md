# React-Router Configuration

1. To configure route with our react application, React-router provides `BrowserRouter`, we need to wrap `BrowserRouter` with entire component
    
```jsx title='React-router setup index.js'

ReactDOM.render(
    <React.StrictMode>
        <BrowserRouter>
            <App />
        </BrowserRouter>
    </React.StrictMode>
)
```
